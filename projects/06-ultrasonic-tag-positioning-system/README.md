# Ultrasonic Tag Positioning System / 超声波标记定位系统

## One-line Summary / 一句话概述

A real-time monitoring and positioning system for ultrasonic tags, covering signal decoding, depth calculation, clock synchronization, TDOA positioning, and web visualization.

一个超声波标记实时监测与定位系统，覆盖信号解码、深度计算、接收机校时、TDOA 定位和 Web 可视化。

## Core Problem / 核心问题

Ultrasonic tag positioning is not a single algorithm problem. It requires stable integration of hardware, receivers, signal filtering, clock synchronization, positioning algorithms, database, and visualization.

超声波标记定位不是单一算法问题，而是硬件、接收机、信号筛选、校时、定位算法、数据库和可视化的稳定集成问题。

## Core Innovation / 核心创新

Integrate depth calculation, valid signal filtering, receiver clock synchronization, TDOA positioning, and web tracking into one engineering workflow.

将深度解算、有效信号筛选、接收机校时、TDOA 定位和 Web 轨迹展示整合为一条工程化流程。
