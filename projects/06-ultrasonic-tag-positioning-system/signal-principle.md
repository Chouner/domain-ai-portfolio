# Signal Principle / 信号原理

## Tag Signal / 标记信号

- Ultrasonic tag emits acoustic pulses around 60 kHz / 超声波标记发射约 60 kHz 声脉冲
- Sensor values are encoded as time intervals between pulses / 传感器数值通过脉冲间隔编码
- Double-pulse mode can encode depth / 双脉冲模式可编码深度
- Triple-pulse mode can encode depth and temperature / 三脉冲模式可编码深度和温度

## Receiver Output / 接收机输出

- tag_id / 标记 ID
- receiver_id / 接收机 ID
- frequency / 频率
- arrival_time / 到达时间
- correlation / 相关性
