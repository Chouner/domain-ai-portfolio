# Skill Standard / Skill 标准

## Skill Folder Structure / Skill 目录结构

```text
skill-name/
├── skill.md
├── references.md
├── input-contract.md
├── output-contract.md
├── pseudocode.md
├── test-cases.md
├── evaluation.md
└── known-issues.md
```

## Required Fields / 必填字段

| Field / 字段 | Description / 说明 |
|---|---|
| Purpose / 目标 | What this skill solves / 这个 Skill 解决什么问题 |
| Domain Formula / 领域公式 | Physical or algorithmic basis / 物理或算法依据 |
| Input Contract / 输入契约 | Required data format / 必要数据格式 |
| Output Contract / 输出契约 | Expected result format / 预期结果格式 |
| Parameters / 参数 | Valid ranges and dependencies / 有效范围和依赖关系 |
| Upstream Dependencies / 上游依赖 | Required previous modules / 前置模块 |
| Downstream Usage / 下游用途 | Which modules can use the output / 哪些模块会使用输出 |
| Evaluation / 评估 | How to verify correctness / 如何验证正确性 |
| Limitations / 限制 | When not to use / 何时不应使用 |
