# Knowledge Graph Design / 知识图谱设计

## Node Types / 节点类型

- Formula / 公式
- Algorithm module / 算法模块
- Input data type / 输入数据类型
- Output data type / 输出数据类型
- Parameter / 参数
- Reference source / 参考来源
- Test case / 测试样例
- Known issue / 已知问题

## Edge Types / 边类型

- depends_on / 依赖
- generates / 生成
- validates / 验证
- requires / 需要
- can_feed_into / 可作为下游输入
- conflicts_with / 冲突
