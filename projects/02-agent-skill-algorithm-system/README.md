# Agent + Skill Acoustic Algorithm System / Agent + Skill 声学算法系统

## One-line Summary / 一句话概述

A skill-based algorithm engineering system that converts acoustic formulas, expert rules, input/output contracts, and algorithm dependencies into reusable AI-callable modules.

一个 Skill 化算法工程系统，将声学公式、专家规则、输入输出契约和算法依赖转化为可复用、可被 AI 调用的模块。

## Core Problem / 核心问题

In acoustic algorithm development, domain experts and software developers often lose physical constraints during implementation. This leads to outputs that may run successfully but are difficult to trust or debug.

在声学算法开发中，领域专家和软件开发之间容易丢失物理约束，导致程序虽然能运行，但结果难以信任或调试。

## Core Innovation / 核心创新

Build a physical-constraint layer before algorithm implementation, so every AI-called Skill is bound to formulas, parameter ranges, input/output contracts, upstream/downstream dependencies, test cases, and known limitations.

在算法实现前建立物理约束层，使每个被 AI 调用的 Skill 都绑定公式、参数范围、输入输出契约、上下游依赖、测试样例和已知限制。

## Key Evidence / 关键证据

- About 29 candidate acoustic processing modules identified / 识别约 29 个候选声学处理模块
- 4 modules explicitly organized as Skill manifests / 4 个模块已明确组织为 Skill 清单
- Designed for historical sonar data processing context / 面向历史声呐数据处理场景设计
