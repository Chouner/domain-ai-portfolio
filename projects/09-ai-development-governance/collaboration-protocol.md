# Collaboration Protocol / 协作协议

## Before Coding / 编码前

- Confirm task scope / 确认任务范围
- Read current architecture / 阅读当前架构
- Identify files to modify / 确认需要修改的文件
- Define acceptance criteria / 定义验收标准

## During Coding / 编码中

- Modify only related files / 只修改相关文件
- Keep change log / 保留变更记录
- Ask before architecture-level changes / 架构级变更前先确认
- Add tests where possible / 尽可能补充测试

## After Coding / 编码后

- Summarize changes / 总结变更
- Report risks / 报告风险
- Run tests / 运行测试
- Provide rollback suggestion / 提供回滚建议
