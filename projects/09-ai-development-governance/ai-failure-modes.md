# AI Failure Modes / AI 失败模式

- Misunderstanding user requirement / 误解用户需求
- Losing long-term context / 丢失长期上下文
- Changing unrelated code / 修改无关代码
- Overwriting previous decisions / 覆盖已有决策
- Inventing unavailable APIs / 编造不可用 API
- Ignoring test constraints / 忽略测试约束
- Multi-agent conflict / 多 Agent 冲突
- Lack of acceptance criteria / 缺少验收标准
