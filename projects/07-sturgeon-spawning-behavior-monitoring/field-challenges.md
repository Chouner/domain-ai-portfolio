# Field Challenges / 野外挑战

- Receiver reboot during long-term monitoring / 长期监测中接收机重启
- Data continuity after power loss / 断电后的数据连续性
- Multipath interference / 多路径干扰
- Clock synchronization across receivers / 多接收机校时
- Unstable field deployment conditions / 不稳定现场布设条件
- Difficulty validating positioning results / 定位结果验证困难
