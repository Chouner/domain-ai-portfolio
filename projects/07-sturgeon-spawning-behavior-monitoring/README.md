# Sturgeon Spawning Behavior Monitoring / 中华鲟产卵行为监测

## One-line Summary / 一句话概述

A field-oriented acoustic monitoring project using ultrasonic tags to study sturgeon spawning behavior under long-term monitoring conditions.

一个面向野外长期监测场景的声学监测项目，使用超声波标记研究中华鲟产卵行为。

## Core Problem / 核心问题

Long-term field monitoring faces receiver reboot, data discontinuity, clock drift, multipath interference, and uncertain deployment conditions.

长期野外监测会遇到接收机重启、数据不连续、时钟漂移、多路径干扰和布设条件不确定等问题。

## Core Value / 核心价值

This project demonstrates the ability to combine field deployment, hardware constraints, embedded optimization, signal filtering, clock synchronization, and acoustic positioning into a deliverable monitoring workflow.

该项目体现了将现场布设、硬件约束、嵌入式优化、信号筛选、校时和声学定位整合为可交付监测流程的能力。
