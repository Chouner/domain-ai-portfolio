# Acoustic QA Agent / 声学问答 Agent

## One-line Summary / 一句话概述

A knowledge-base QA Agent for answering basic underwater acoustics, sonar product, and application-scenario questions through a WeChat official account.

一个基于知识库的声学问答 Agent，用于在微信公众号等场景中回答基础水声、声呐产品和应用场景问题。

## Core Problem / 核心问题

Sonar products have a high knowledge barrier. Users often ask repetitive questions about sonar principles, product features, parameters, and application scenarios.

声呐产品有较高知识门槛，用户经常反复询问声呐原理、产品功能、参数含义和应用场景。

## Core Innovation / 核心创新

Convert scattered acoustic knowledge from books, exams, papers, product documents, and expert explanations into a deployable AI customer-service knowledge base.

将书籍、考试题、论文、产品文档和专家解释中分散的声学知识，转化为可检索、可问答、可部署的 AI 客服知识库。

## My Role / 我的角色

- 100 core acoustic terms / 100 个核心声学术语
- 3789 knowledge descriptions / 3789 条知识描述
- Qwen-7B fine-tuning validation / Qwen-7B 微调验证
- 6-month trial deployment / 6 个月试运行
- 218 users served / 服务 218 位用户

## Limitations / 边界

Pricing, customized engineering scenarios, and project-specific parameter selection still require human follow-up.

价格、定制化工程场景和项目级参数选择仍然需要人工跟进。
