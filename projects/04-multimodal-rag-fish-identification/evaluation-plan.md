|---|---|

| Hallucination rate / 幻觉率 | Check unsupported species claims / 检查无来源支持的物种判断 |
| Response latency / 响应延迟 | Measure API response time / 测量 API 响应时间 |
| Similar-species confusion / 相似物种混淆 | Record confusion pairs / 记录混淆物种对 |
