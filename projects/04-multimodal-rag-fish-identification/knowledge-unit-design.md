# Knowledge Unit Design / 知识单元设计

## Problem / 问题

Raw document chunks were fragmented. Retrieval often returned incomplete context.

原始文档切片过于碎片化，检索结果经常缺少完整上下文。

## Solution / 方案

Semantic aggregation was used to reconstruct 880 fragmented text chunks into 90 complete knowledge units.

通过语义聚合，将 880 个碎片化文本块重构为 90 个完整知识单元。

## Knowledge Unit Fields / 知识单元字段

- Species name / 物种名
- Morphological features / 形态特征
- Habitat / 栖息地
- Similar species / 相似物种
- Identification cues / 识别线索
- Reference text / 参考文本
- Image-related clues / 图像相关线索
