# Baseline Analysis / 基线分析

## Traditional Route / 传统路线

- YOLO-style detection/classification / YOLO 风格检测或分类
- CLIP-style image-text matching / CLIP 风格图文匹配

## Observed Problem / 观察到的问题

Early trials on low-quality, small-sample deep-sea fish images showed poor performance, with accuracy around 20% in the exploratory stage.

在低质量、小样本深海鱼类图像上的早期试验表现较差，探索阶段准确率约 20%。

## Reason for Route Change / 路线调整原因

The problem required expert knowledge about morphology, habitat, body shape, color pattern, and similar species. Therefore, a knowledge-enhanced multimodal RAG route was explored.

该问题需要形态、栖息地、体型、颜色模式和相似物种等专家知识，因此转向知识增强的多模态 RAG 路线。
