# Technology Scoring Model / 技术评分模型

| Dimension / 维度 | Description / 说明 |
|---|---|
| Technical maturity / 技术成熟度 | Is it research, demo, or production ready? / 属于研究、Demo 还是生产可用？ |
| Business relevance / 业务相关性 | Does it match current company scenarios? / 是否匹配当前业务场景？ |
| Implementation cost / 实施成本 | Data, compute, engineering resources / 数据、算力和工程资源 |
| Risk / 风险 | Security, hallucination, reliability / 安全、幻觉和可靠性 |
| Reusability / 复用性 | Can it be reused across projects? / 是否可跨项目复用？ |
| Priority / 优先级 | Short-term, mid-term, or long-term / 短期、中期或长期 |
