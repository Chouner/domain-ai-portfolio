# 声呐自动处理指标

## 1. 数据处理指标

| 指标 | 计算方式 | 用途 |
|---|---|---|
| 文件成功率 | `processed_raw_files / total_raw_files` | 批处理可靠性 |
| 产物完整度 | `generated_required_artifacts / required_artifacts` | 交付准备度 |
| 有效样本比例 | `valid_cells / total_cells` | 数据质量 |
| 单 raw 运行时间 | `total_runtime / raw_file_count` | 吞吐能力 |
| 人工介入次数 | 阻塞性人工事件计数 | 自动化程度 |

## 2. 算法精度指标

| 阶段 | 指标 | 目标 / 解释 |
|---|---|---|
| Raw-to-Sv | RMSE、MAE、p95 绝对误差、最大绝对误差 | RMSE < 0.5 dB |
| 去噪 | SNR 提升、mask IoU、误删率 | IoU > 0.85 |
| 表层/底线 | 米级 MAE、p95 误差、断裂数 | 底线误差 < 1 m |
| NASC/EDSU | 相对误差、有效 cell 比例 | 相对误差 < 5% |
| AI 预审 | precision、recall、误报率、人工一致性 | 人工认可率 > 90% |

## 4. 证据边界

- 38 kHz raw-to-Sv 对比作为已验证证据。
- 历史 CSV 清洗和底线检测实验只作为探索证据。
- 背景噪声和中值滤波对比需要在修正 no-data 语义后重新验证；之前的 `-999` 假设不能作为最终证据。
- 最终生产声明应包含数据集版本、频率、校准假设、no-data 定义、参数集、参考导出和验证脚本 hash。
