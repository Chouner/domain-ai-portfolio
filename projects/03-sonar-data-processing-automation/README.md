# Sonar Data Processing Automation / 声呐数据处理自动化

## One-line Summary / 一句话概述

An AI-assisted workflow that selects and orchestrates sonar data processing modules based on data type, data quality, and analysis goal.

一个 AI 辅助工作流，根据数据类型、数据质量和分析目标选择并编排声呐数据处理模块。

## Core Problem / 核心问题

Sonar data processing usually depends on expert judgment. Different devices, data quality, and analysis goals require different processing pipelines.

声呐数据处理通常依赖专家判断。不同设备、数据质量和分析目标需要不同处理流程。

## Core Innovation / 核心创新

Transform fixed-script processing into an AI decision workflow: data profiling -> module selection -> parameter confirmation -> processing -> validation -> result generation.

将传统固定脚本处理改造成 AI 决策工作流：数据画像 -> 模块选择 -> 参数确认 -> 处理执行 -> 结果验证 -> 结果生成。
