# Role-based Project Storytelling / 岗位化项目表达

The same project should be described differently for different roles.

同一个项目应该根据不同岗位采用不同表达重点。

## AI Product Manager / AI 产品经理

Focus / 重点：

- User problem / 用户问题
- Product decision / 产品决策
- MVP / MVP
- Evaluation metric / 评估指标
- Business value / 业务价值

## AI Application Engineer / AI 应用工程师

Focus / 重点：

- Architecture / 架构
- Data pipeline / 数据流程
- Model workflow / 模型工作流
- Engineering implementation / 工程实现
- Metrics / 指标

## AI Solution Product Manager / AI 解决方案产品经理

Focus / 重点：

- Customer scenario / 客户场景
- Workflow automation / 工作流自动化
- PoC to deployment / 从 PoC 到部署
- Adoption / 采用情况
- Reusability / 复用性

## Algorithm Engineering / 算法工程

Focus / 重点：

- Algorithm logic / 算法逻辑
- Data quality / 数据质量
- Validation / 验证
- Performance / 性能
- Boundary conditions / 边界条件
