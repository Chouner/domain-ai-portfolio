# Confidentiality Statement / 保密声明

This repository is a sanitized portfolio version.

本仓库是经过脱敏处理的公开作品集版本。

## It does not include / 不包含

- Proprietary source code / 公司核心源码
- Internal algorithms / 内部算法
- Confidential datasets / 保密数据集
- Customer information / 客户信息
- Production credentials / 生产凭据
- Internal business documents / 内部业务文档
- Non-public product specifications / 未公开产品规格

## It includes / 包含

- Anonymized architecture / 脱敏架构
- Simplified workflows / 简化工作流
- Pseudocode / 伪代码
- Public or synthetic examples / 公开或合成示例
- Non-sensitive metrics / 非敏感指标
- Evaluation methods / 评估方法
- Personal reflections on system design / 个人系统设计复盘

The goal is to demonstrate project thinking, system design, evaluation awareness, and domain-AI delivery capability without exposing sensitive organizational information.

本仓库的目标是在不暴露组织敏感信息的前提下，展示项目思维、系统设计、评估意识和专业领域 AI 交付能力。
