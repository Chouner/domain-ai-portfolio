# Domain AI Portfolio

This repository documents selected AI application and algorithm-engineering projects in domain-specific scenarios, especially acoustic data processing, sonar image understanding, RAG, Agent workflows, and skill-based algorithm systems.

本仓库记录专业领域 AI 应用与算法工程化项目，重点展示声学数据处理、声呐图像理解、RAG、Agent 工作流和 Skill 化算法系统。

## Focus Areas / 重点方向

- Domain-specific RAG and knowledge base systems / 专业领域 RAG 与知识库系统
- Agent + Skill workflow design / Agent + Skill 工作流设计
- Multimodal sonar image understanding / 多模态声呐图像理解
- AI-assisted annotation and detection / AI 辅助标注与检测
- Acoustic algorithm engineering / 声学算法工程化
- Evaluation and traceability design / 评估体系与可追溯设计

## Confidentiality / 保密边界

All projects are documented with anonymized data, simplified architecture, pseudocode, and non-sensitive metrics. Proprietary algorithms, production source code, customer data, and confidential business information are not included.

所有项目均使用脱敏数据、简化架构、伪代码和非敏感指标进行记录。不包含公司核心算法、生产源码、客户数据或商业机密。
