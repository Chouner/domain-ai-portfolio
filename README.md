# Domain AI Portfolio

A portfolio of domain-specific AI systems, focusing on RAG, Agent workflows, multimodal AI, sonar data processing, and acoustic algorithm engineering.

## Focus Areas
- Domain-specific RAG and knowledge base systems
- Agent + Skill workflow design
- Multimodal sonar image understanding
- AI-assisted annotation and detection
- Acoustic algorithm engineering
- Evaluation and traceability design

## Selected Projects

| Project | Focus | Key Evidence |
|---|---|---|
| Acoustic QA Agent | RAG / Knowledge Base / AI Customer Service | 100 core terms, 3789 knowledge descriptions, 6-month trial, 218 users |
| Agent + Skill Algorithm System | Agent Workflow / Algorithm Engineering | Skill-based architecture, acoustic algorithm constraints |
| Sonar Data Processing Automation | AI Decision Workflow | Modular processing pipeline |
| Multimodal RAG Fish Identification | Multimodal RAG / Small-sample Recognition | 880 text chunks reconstructed into 90 knowledge units |
| ARIS Jellyfish Detection | Multimodal Annotation / Object Detection | 86,342 ARIS frames, 74,092 detection frames |
| Ultrasonic Tag Positioning | TDOA / Clock Sync / Acoustic Positioning | 4,720 pulse groups, 25,064 TDOA pairs, 21,292 depth values |
